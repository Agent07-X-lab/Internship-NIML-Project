Co-activation Edge Weight Formula (Jaccard Similarity)

Edge weights between appliance node \(i\) and node \(j\) represent their co-occurrence frequency, calculated using the Jaccard Similarity of their binary ON/OFF state sequences (\(S_i\), \(S_j\)) over the sliding window:

$$A_{i,j} = \text{Jaccard}(S_i, S_j) = \frac{|S_i \cap S_j|}{|S_i \cup S_j|} = \frac{\sum_{t} (S_{i,t} \cdot S_{j,t})}{\sum_{t} S_{i,t} + \sum_{t} S_{j,t} - \sum_{t} (S_{i,t} \cdot S_{j,t})}$$
Variables:
  • \(A_{i,j}\): Co-activation edge weight between appliance node \(i\) and appliance node \(j\).
  • \(S_i, S_j\): The binary status sequences (ON/OFF) of appliance \(i\) and appliance \(j\) over the window.
  • \(S_{i,t}, S_{j,t}\): The active status (1 for ON, 0 for OFF) of appliance \(i\) and appliance \(j\) at time step \(t\).
  • \(|S_i \cap S_j|\): Intersection of active states (time steps where both appliances are active concurrently).
  • \(|S_i \cup S_j|\): Union of active states (time steps where at least one appliance is active).

Spatiotemporal Benchmark baselines & Formulas

The anomaly diagnostics utilize a **Graph Transformer Autoencoder (GTAE)** framework to learn a multi-dimensional normal baseline representation of electrical loads.

Joint Network Loss Function

The multi-task optimization function optimized by the autoencoder model:

$$\mathcal{L}_{\text{total}} = \mathcal{L}_{X} + 0.2 \cdot \mathcal{L}_{A}$$
Variables:
  • \(\mathcal{L}_{\text{total}}\): Joint optimization loss of the Graph Transformer Autoencoder.
  • \(\mathcal{L}_{X}\): Mean Squared Error (MSE) loss for node continuous feature (power level) reconstruction.
  • \(\mathcal{L}_{A}\): Binary Cross-Entropy (BCE) loss for pairwise structural network connectivity reconstruction.
  • \(0.2\): Weighting factor balancing the feature reconstruction and structure reconstruction tasks.

Graph Delta Anomaly Metric

The structural divergence of the appliance network relative to baseline operating standards:

$$\Delta_t = \| A_t - A_{\text{baseline}} \|_F$$
Variables:
  • \(\Delta_t\): Anomaly score at time step/sliding window \(t\).
  • \(A_t\): Appliance state Jaccard adjacency matrix for window/time step \(t\).
  • \(A_{\text{baseline}}\): Learned baseline normal co-occurrence adjacency matrix of the home.
  • \(\| \cdot \|_F\): Frobenius norm (square root of the sum of the absolute squares of all matrix elements).
